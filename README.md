# AERO NDT

Site web moderne pour [AERO NDT](https://aero-ndt.com) — prestations en Contrôle Non Destructif pour la filière aéronautique.

## Stack

- [Next.js 16](https://nextjs.org/) (App Router)
- [Tailwind CSS 4](https://tailwindcss.com/)
- [Lucide React](https://lucide.dev/) (icônes)
- Déployé sur [Vercel](https://vercel.com/)

## Développement

```bash
npm install
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Structure

- `src/app/` — Pages (Accueil, À propos, Services, Contact, Carrières)
- `src/components/` — Composants réutilisables
- `src/lib/site-config.ts` — Configuration et contenu du site
